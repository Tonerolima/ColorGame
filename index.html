About RGB Color Code


There are 3 primary colors, Red Green Blue (RGB), which when added in varying quantities produces a broad array of colors.

For instance, when Red is added to Green we get Yellow and when Green is combined with Blue we get Cyan. Mixing all 3 colors produces White as seen in the image below. In contrast, no colors = black.

Mixing equal amounts of the primary colors produce basic secondary colors but as we know, there are usually several shades of the same color. For instance wine is brighter than horse-blood even though they are both labelled 'Red'.

Try changing the values in the text boxes below and observe the resulting color in the square.

This varying degree of intensity is the result of mixing uneven amounts of the primary colors. Each color in the RGB spctrum has a range of 0 to 255, so changing those values result in very unique colors.

The rules of the game is simple and straightforward. All you have to do is select the box that correctly displays the color that corresponds with the rgb color code on the top of the page.
